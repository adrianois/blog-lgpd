# Blog LGPD

Um blog dedicado à Lei Geral de Proteção de Dados (LGPD) no Brasil, oferecendo conteúdo educativo, notícias, cursos e recursos para profissionais e cidadãos interessados em privacidade e proteção de dados.

## 📋 Sobre

Este projeto contém o código fonte do site estático do Blog LGPD, desenvolvido com HTML, CSS e JavaScript puro. O foco é fornecer informações acessíveis e práticas sobre a LGPD (Lei nº 13.709/2018).

## 🚀 Funcionalidades

- **Conteúdo Educativo**: Artigos explicativos sobre LGPD
- **Notícias e Jurisprudência**: Atualizações sobre decisões e regulamentações
- **Glossário**: Termos técnicos explicados
- **Downloads**: Recursos gratuitos para download
- **Cursos**: Informações sobre formações disponíveis
