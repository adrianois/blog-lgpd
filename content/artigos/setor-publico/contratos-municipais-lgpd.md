# Cláusulas LGPD obrigatórias em contratos municipais

Modelo de cláusulas de proteção de dados para incluir em contratos e licitações da administração pública.

## Obrigatoriedade das cláusulas

### Base legal
- Lei Geral de Proteção de Dados (LGPD)
- Lei de Licitações (Lei 8.666/1993)
- Princípios da administração pública

### Quando incluir
- Contratos que envolvam dados pessoais
- Licitações com tratamento de informações
- Parcerias público-privadas

## Cláusulas essenciais

### 1. Definições
```
Os termos abaixo terão os seguintes significados:
- "Dados Pessoais": qualquer informação que identifique ou torne identificável uma pessoa natural
- "Tratamento": qualquer operação realizada com dados pessoais
- "Titular": pessoa natural a quem se referem os dados pessoais
```

### 2. Obrigações gerais
```
O CONTRATADO obriga-se a:
a) Tratar os dados pessoais em conformidade com a LGPD
b) Implementar medidas de segurança adequadas
c) Não utilizar os dados para fins diversos do contrato
d) Comunicar imediatamente qualquer incidente de segurança
```

### 3. Responsabilidade solidária
```
O CONTRATADO será responsável solidariamente com o CONTRATANTE por eventuais violações à LGPD decorrentes do tratamento de dados pessoais no âmbito deste contrato.
```

### 4. Subcontratação
```
Qualquer subcontratação que envolva tratamento de dados pessoais dependerá de prévia autorização escrita do CONTRATANTE e inclusão das mesmas cláusulas de proteção de dados.
```

### 5. Segurança da informação
```
O CONTRATADO implementará as seguintes medidas mínimas de segurança:
a) Criptografia de dados em trânsito e repouso
b) Controle de acesso baseado em funções
c) Logs de auditoria
d) Plano de resposta a incidentes
```

### 6. Direitos dos titulares
```
O CONTRATADO facilitará o exercício dos direitos dos titulares previstos na LGPD, incluindo:
a) Acesso aos dados
b) Correção de informações
c) Exclusão de dados
d) Portabilidade
```

### 7. Comunicação de incidentes
```
O CONTRATADO comunicará ao CONTRATANTE e à ANPD qualquer incidente de segurança envolvendo dados pessoais no prazo máximo de 72 (setenta e duas) horas.
```

### 8. Auditoria e fiscalização
```
O CONTRATANTE poderá auditar as medidas de proteção de dados implementadas pelo CONTRATADO, que se obriga a fornecer toda documentação necessária.
```

### 9. Penalidades
```
O descumprimento das cláusulas de proteção de dados sujeitará o CONTRATADO às seguintes penalidades:
a) Multa de [valor] por infração
b) Rescisão unilateral do contrato
c) Responsabilidade civil por danos causados
```

### 10. Vigência pós-contratual
```
As obrigações de proteção de dados permanecerão vigentes mesmo após o término do contrato, pelo prazo necessário para eliminação segura dos dados.
```

## Modelo completo de cláusula

### Cláusula de Proteção de Dados
```
CLÁUSULA [NÚMERO] - PROTEÇÃO DE DADOS PESSOAIS

1. O CONTRATADO declara ciência e concorda em cumprir integralmente a Lei Geral de Proteção de Dados (Lei nº 13.709/2018) e suas normas regulamentares.

2. O CONTRATADO obriga-se a tratar os dados pessoais coletados no âmbito deste contrato de forma lícita, adequada, necessária e transparente.

3. Ficam estabelecidas as seguintes obrigações específicas:
   [Listar obrigações detalhadas]

4. O CONTRATADO será responsável por eventuais violações à legislação de proteção de dados, respondendo solidariamente com o CONTRATANTE.

5. Esta cláusula permanecerá em vigor durante toda a execução do contrato e por [prazo] após seu término.
```

## Aplicação em licitações

### Edital
- Incluir cláusula específica no edital
- Exigir declaração de conformidade
- Avaliar propostas com foco em segurança

### Contrato
- Detalhar obrigações específicas
- Definir métricas de conformidade
- Estabelecer mecanismos de fiscalização

## Desafios comuns

### Contratos legados
- Revisão de contratos existentes
- Aditivo contratual
- Adequação gradual

### Pequenos fornecedores
- Capacitação técnica
- Modelos simplificados
- Acompanhamento próximo

## Fiscalização e penalidades

### ANPD
- Multas por descumprimento
- Obrigatoriedade de adequação
- Publicização de infrações

### Administração pública
- Responsabilidade do gestor
- Controle interno
- Prestação de contas
